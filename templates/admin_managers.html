About Manager Role
Manager Permissions:
  • Access to view all uploaded documents
  • Ability to decrypt and access original document content
  • Cannot upload or redact documents (customer function)
  • Cannot manage user accounts (admin function)
Security Notes:
  • Managers can see PII data in documents
  • All document accesses are logged for audit
  • Create manager accounts only for authorized personnel
